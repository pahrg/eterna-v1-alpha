## Hur hittar jag rätt information?

- **Ny användare?** Börja med [Snabbstart](#theme/usage/Quickstart_sv_SE.md)
- **Söker efter något specifikt?** Använd den [Avancerade Sökningen](usage/Advanced_Search.md)
- **Har problem?** Kolla [Felsökning](usage/Troubleshooting.md) eller [Vanliga Frågor](usage/FAQ.md)
- **Administratör?** Se avsnittet Administrationsguider ovan
- **Utvecklare?** Börja med [Utvecklarguide](development/Developers_Guide.md)

## Behöver du mer hjälp?

Om du inte hittar det du söker kan du:
- Kontakta vårt supportteam på earkiv@whitered.se
- Skapa ett ärende på [GitHub](https://github.com/ETERNA-earkiv/ETERNA/issues)
- Delta i diskussioner på våra [forum](http://eterna.whitered.se)

ETERNA är designat för att vara användarvänligt och kraftfullt. Vi hoppas att denna dokumentation hjälper dig att få ut det mesta av systemet!
